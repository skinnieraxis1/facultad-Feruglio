## Project Lifecycle Commands
# Basics
- `stack new <proyect-name>`: Creamos un nuevo proyecto
- `stack setup`
- `stack build`
- `stack test`: Corremos los archivos test
- `stack clean`
# Execution
- `stack ghci`: Levantamos el proyecto de haskell
- `stack ghci file.hs`: Levantamos un archivo específico de haskell
